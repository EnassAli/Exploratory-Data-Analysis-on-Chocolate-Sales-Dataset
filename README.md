# Exploratory-Data-Analysis-on-Chocolate-Sales-Dataset
In this lab, the Chocolate Sales dataset was used to perform Exploratory Data Analysis (EDA).

The dataset contains sales transactions with the following features:
- Sales Person
- Country
- Product
- Date
- Amount
- Boxes Shipped

The goal of this analysis is to explore the dataset, clean it, visualize it, and extract meaningful insights.

# Data Visualization
Several visualizations were created:
- Bar chart for the most sold products
- Count plot for sales by country
- Histogram for sales amount distribution
- Scatter plot for the relationship between boxes shipped and sales amount

# Insights
The product that generated the highest total sales is Smooth Sliky Salty.
Australia has the highest number of sales transactions.
Australia also achieved the highest total sales amount.
Most sales amounts fall between 2,500 and 8,500, which represents the typical transaction range.
There is no strong relationship between the number of boxes shipped and the sales amount.
